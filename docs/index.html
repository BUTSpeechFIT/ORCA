ORCA is a framework for assessing the quality of open-ended responses in audio question-answering tasks. It uses language model representations with Beta distribution modeling to capture both the mean quality and uncertainty (variance) of responses based on multiple human annotations.

üéØ Multiple Scoring Functions

Support for Bernoulli, Beta NLL, Beta Moment Matching (BMM), and MSE loss functions.

üìä Uncertainty Estimation

Models annotation variance using Beta distribution parameters (Œ±, Œ≤) to capture disagreement.

‚ö° Efficient Fine-tuning

Uses LoRA (Low-Rank Adaptation) for parameter-efficient training on large language models.

üîÑ Distributed Training

Built on HuggingFace Accelerate for seamless multi-GPU training.

üèóÔ∏è Model Architecture

ORCA uses a pre-trained language model (e.g., Gemma, Llama, OLMo) with a linear scoring head that outputs log(Œ±) and log(Œ≤) parameters for a Beta distribution.

Beta Distribution captures:
  • Mean quality: E[score] = Œ± / (Œ± + Œ≤)
  • Uncertainty/Variance: Var[score] = (Œ±¬∑Œ≤) / ((Œ±+Œ≤)¬≤¬∑(Œ±+Œ≤+1))

üöÄ Quick Start

Installation

# Python 3.12+ required
pip install -e .

Training

python -m orca_score.train \
    --train_data path/to/train.json \
    --val_data path/to/val.json \
    --model google/gemma-3-1b-it \
    --score_type beta \
    --lora_rank 256 \
    --output_dir ./output \
    --log_dir ./logs

Inference

python -m orca_score.infer \
    --model_path ./output/best \
    --data path/to/test.json \
    --output_dir ./results
